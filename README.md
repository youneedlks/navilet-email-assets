# Навылет! AI — ассеты для письма

Картинки для `navilet_email_v2.html`. Письмо ссылается на них через jsDelivr CDN.

## Как опубликовать (один раз, ~3 минуты)

1. Создайте **новый публичный** репозиторий на GitHub с именем **`navilet-email-assets`**
   (под аккаунтом `youneedlks` — тогда URL в письме совпадут «из коробки»).
2. Залейте СОДЕРЖИМОЕ этой папки (`logos/`, `photos/`, `icons/`, этот README) в корень репозитория.
   Проще всего: на странице репозитория → **Add file → Upload files** → перетащить папки → **Commit**.
3. Готово. Через ~1 минуту картинки начнут отдаваться с CDN по адресам вида:
   `https://cdn.jsdelivr.net/gh/youneedlks/navilet-email-assets@main/logos/mgimo.png`

Если выберете другое имя/аккаунт — пришлите его, заменю базовый URL в письме
(он один: `https://cdn.jsdelivr.net/gh/youneedlks/navilet-email-assets@main`).

## Состав

- `logos/` — навылет-марка, МГП, AnyTour, МГИМО, ТПП, РСТ
- `photos/` — Барзыкин, Агафонов, Осауленко (квадрат 200×200)
- `icons/` — line-иконки (lucide) в фирменном цвете для блоков «возможности», «4 шага», «каналы»

Фото отелей в виджете грузятся напрямую с `static.tourvisor.ru` и не требуют хостинга.
